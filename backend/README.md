# How to run
### Run with IntelliJ
- Main function
`src/main/java/com.piay.covid19.springboot/web/Application`

### Using CLI on Mac
```shell
$ brew install gradle
$ gradle wrapper --gradle-version 4.10
$ ./gradlew build
$ ./gradlew bootRun
```
